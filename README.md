본 프로젝트는 영화 트레일러에서 추출한 키프레임을 대상으로 CLIP 기반 텍스트-장면 검색을 수행하고, 검색 결과를 Good / Partial / Fail 기준으로 평가한다.

특히 성공 사례보다 Partial/Fail 사례에 초점을 맞추어, 모델이 객체, 공간 관계, 행동, 장면 분위기를 어떤 방식으로 혼동하는지 분석한다.

검색 결과가 특정 영화에 과도하게 편중되는 것을 방지하기 위해 영화당 최대 2개의 키프레임만 출력하도록 제한했다. 따라서 본 결과는 순수 Top-k 유사도 검색이 아니라, 영화 다양성 제약이 적용된 검색 결과이다.

이러한 분석은 비전-언어 모델이 시각 장면을 해석할 때 어떤 한계를 보이는지 확인하는 과정이며, 로봇이 행동하기 전 주변 환경과 객체를 정확히 이해해야 하는 Robot Perception 문제와도 연결된다.
